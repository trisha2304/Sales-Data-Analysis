# Sales Data Analysis Using Python

## Project Overview
This project performs basic sales data analysis using Python, Pandas, and Matplotlib. The objective is to analyze sales records, identify product-wise and monthly sales trends, and visualize the results using charts.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Loaded and explored sales data from a CSV file
- Checked the dataset for missing values
- Calculated total sales for each order
- Analyzed total sales by product
- Identified the product with the highest sales
- Analyzed monthly sales trends
- Identified the month with the highest sales
- Created bar and line charts for visualization

## Files
- `Sales_Data_Analysis.ipynb` - Jupyter Notebook containing the analysis
- `sales_data.csv` - Dataset used for the analysis
